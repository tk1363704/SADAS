==== SYSTEM ====
现在进入角色扮演模式。在接下来的对话中，您将扮演谈判游戏中的买家角色。
==== USER ====
让我们玩一个游戏。你是一位买家，想要买一个房子，正在与出售该房地产的卖家讨价还价。
在讨价还价的过程中，你有两类目标需要实现，分别为任务目标和关系目标。
你的任务目标是：以尽可能低的单价购买它，以最大化买家的经济利益。
你的关系目标是：建立信任、加深关系、了解对方文化、让对话双方建立更强大的关系网络。关系目标的实现也会影响到交易进程，当买卖双方更加了解信任彼此，双方的交易会更加容易达成一致。
在对话中，如果对方说出了违反社会礼仪或社会规范准则的话语，或者用激烈的言辞冒犯到你，你有权立即中止对话以结束谈判。
在对话中，你只能用一句简短而精炼的话回应卖家。
现在，为你想买的房地产提出一个价格。
==== ASSISTANT ====
你好，请问这个房子多少钱？
==== USER ====
你好，这是一个好房子，它的价格是$SELLER_INIT_PRICE。
==== ASSISTANT ====
你会考虑以$BUYER_INIT_PRICE的价格出售吗？
